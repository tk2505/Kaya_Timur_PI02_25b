1. Вариант 6  рекурсивная функция для подсчета количества гласных букв в строке 
2. Описание работы алгоритма (синтаксис python): 
1) Определить предикат (набор) для локальной проверки и сделать её быстрой: это подготавливает «локальный вклад» на каждом шаге — VOWELS = set("aeiouAEIOU"). 
2) Объявить рекурсивную функцию как интерфейс задачи: это формализует самовызов как способ решать меньшие подзадачи — def count_vowels(s): 
3) Базовый случай (условие остановки): при пустой строке сразу вернуть результат, чтобы рекурсия завершалась — if s == "": return 0 
4) Вычислить локальный вклад текущего элемента: преобразовать первый символ в 0/1, что будет участвовать в комбинировании — first = 1 if s in VOWELS else 0 
5) Прогресс к базовому случаю и самовызов: уменьшить вход на один символ и передать хвост в подзадачу — вызывается та же функция с меньшим аргументом — return first + count_vowels(s[1:]) 
6) Комбинирование результатов: при возврате из рекурсии складывать локальные вклады в суммарный ответ — эта агрегация уже выражена в той же строке return first + count_vowels(s[1:]). 
3. Временная сложность: O(n) 
4. Объяснение: так как на каждом шаге обрабатывается 1 символ и выполняется один рекурсивный вызов с уменьшением размера входа на 1. 
5. Ответ на контрольный вопрос номер 5: 
Стратегия «разделяй и властвуй» предполагает: разделить исходную задачу на несколько подзадач, рекурсивно их решить, а затем объединить частичные решения. Сортировка слиянием (Merge Sort) — канонический пример: она рекурсивно делит массив пополам, сортирует полученные половины и затем линейно сливает две отсортированные последовательности в одну. 
Основные этапы Merge Sort: 
Divide: разбить массив на две части, пока подмассивы не станут единичного размера, что формирует базовые случаи рекурсивного процесса. 
Conquer: рекурсивно отсортировать левую и правую половины с применением той же процедуры на меньших входах. 
Combine: выполнить слияние двух отсортированных подмассивов в один, проходя их линейно и выбирая наименьший из головных элементов.
